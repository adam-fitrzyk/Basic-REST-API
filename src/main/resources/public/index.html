Welcome to this TLPT (TL Practise Task) RESTful API!

Content


  • /users
  • /users/:id
  • /users/search?filter=:filter&filter=:filter&...

  • /events
  • /events/:id
  • /events/search?filter=:filter&filter=:filter&...

    • A filter (:filter) is a json string with the following parameters to filter content retrieved:


        {
        "attribute": text,          >> required
        "operator": text,           >> required
        "value": text or integer,   // optional if "range" field is provided
        "range": {                  // optional if "value" field is provided
            "from": integer,
            "to": integer,
        }
    }

    The operator field must be one of the following:

  • "eq" (meaning equals)
  • "lte" (meaning less than or equal to)
  • "gte" (meaning greater than or equal to)

    • NOTE: if the "range" field is provided, only the "eq" operator can be used.
